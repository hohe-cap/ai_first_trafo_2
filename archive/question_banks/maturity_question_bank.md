# Maturity Question Bank

> **100+ Questions Across 4 Dimensions × 5 Levels (Adaptively Filtered to 20-30)**

---

## Overview

**Purpose:** Diagnose current AI maturity across 4 dimensions and 5 levels

**Dimensions:**
1. **Individual Proficiency** - Personal AI tool usage
2. **Team Workflows** - Collaborative AI integration
3. **Program/Organizational** - Systematic AI adoption
4. **Strategic** - AI-First as competitive advantage

**Maturity Levels:**
- **L0:** No AI usage
- **L1:** Individual Experimentation (ad-hoc tool usage)
- **L2:** Team Integration (workflows designed around AI)
- **L3:** Organizational Scaling (governance, measurement, best practices)
- **L4:** Strategic Innovation (AI-First, competitive moat)

**Adaptive Filtering:** Based on Pre-Readiness + Context, show 20-30 most relevant questions (60-70% reduction)

**Time:** 15-20 minutes

---

## Adaptive Filtering Rules

### Rule 1: Pre-Readiness Based Filtering
```
IF Pre-Readiness < 50%:
  SKIP Level 3 & 4 questions (focus on L1-L2)
  
IF Pre-Readiness 50-70%:
  SHOW Level 1 & 2, selective Level 3
  
IF Pre-Readiness > 70%:
  SHOW all levels
```

### Rule 2: Context-Based Filtering
```
IF Industry = "Banking":
  PRIORITIZE compliance/governance questions (Q3.1, Q3.6)
  SKIP agility questions (Q4.4)
  
IF Team_Size < 10:
  SKIP organizational questions (Q3.1-Q3.10)
  FOCUS on individual/team (Q1.x, Q2.x)
  
IF Compliance_Tier = "Flexible":
  SKIP heavy governance (Q3.1, Q3.6)
  PRIORITIZE velocity (Q2.10, Q4.3)
```

### Rule 3: Response-Adaptive Filtering
```
IF Q1.1 = "Rarely/Never":
  SKIP rest of Level 1 (assume L0)
  
IF Q2.1 < 25%:
  SKIP Level 2 deep-dive (Q2.5-Q2.10)
  
IF Q3.1 = "No governance":
  SKIP governance details (Q3.2-Q3.6)
```

## Scoring Algorithm

**Dimension Score (per level):**
```
Level_Score = Σ(Question_Score × Question_Weight) / Total_Weight
```

**Overall Maturity Level:**
```
IF Level_1_Score > 80%: Maturity >= L1
IF Level_2_Score > 70%: Maturity >= L2
IF Level_3_Score > 60%: Maturity >= L3
IF Level_4_Score > 50%: Maturity = L4
```

**Flight Level Calculation:**
```
Flight_Level = ROUND(
  (Individual × 0.25) + 
  (Team × 0.30) + 
  (Program × 0.25) + 
  (Strategic × 0.20)
)
```

## Industry Variants

### Banking
- +10 questions on compliance (model governance, audit trails)
- +5 questions on legacy integration (COBOL, mainframes)
- Adjusted weights: Governance 40% (vs 25%)

### Healthcare
- +8 questions on HIPAA/PHI handling
- +5 questions on clinical validation
- Adjusted weights: Security 35% (vs 20%)

### Startup
- -15 questions (remove organizational complexity)
- Focus on agility and experimentation
- Adjusted weights: Strategic 30% (vs 20%)
