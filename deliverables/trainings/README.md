# AI Transformation Enablement — Trainingsunterlagen

## Übersicht

10-wöchiges Remote-Training für Agilisten, Agile Enabler und Transformation Manager.
Ziel: Von "AI-interessiert, aber unsystematisch" zu "kann AI-Tools produktiv nutzen UND eine AI-Transformation beim Kunden strukturiert begleiten."
